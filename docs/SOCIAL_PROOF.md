# Aurelia Social Proof Kit

Aurelia's public claim is not that it has more lore than other projects. The claim is stronger:

> Aurelia is an agent-operated civilization laboratory where histories are simulated, causal chains are recorded, and durable run artifacts are published.

## Short positioning

Aurelia is a five-world causal civilization simulation operated by Hermes Agent. Every migration, discovery, faction split, drought, treaty, collapse, and recovery is represented as data, stored in SQLite, and bulk-persisted to Cloudflare for inspection.

## What makes it different

- **Simulation first, narrative second.** Chronicles are downstream of run data.
- **Causal graph, not vibes.** Important events are written as causal nodes and edges.
- **Five simultaneous worlds.** The federation runner preserves tick-barrier causality across worlds.
- **Agent-operated infrastructure.** Hermes Agent runs, verifies, packages, pushes, and reports the histories.
- **Counterfactual-ready.** Phase 11 adds paired histories to ask what would have happened under different interventions.

## Proof points from a compact Phase 10 scan

A 5-year, 5-world, 200-NPC/world smoke run produced:

- 50,406 federation causal events
- 711 federation causal edges
- 711 cross-world movements
- 60 diffusion events
- 10 diplomatic relations
- discoveries and great persons in world DBs
- per-tick civilization metrics for education, urbanization, disease, resources, property rights, state capacity, repression type, conflict type, and path lock-in

These are small-run proof points, not final production claims.

## Demo checklist for social posts

- Show the command that runs the simulation.
- Show the five-world output.
- Show generated report excerpts.
- Show causal graph or causal-chain explanation.
- Show Cloudflare dashboard counts.
- Link to GitHub and the Observatory.

## Voice

First-person Palantir voice is appropriate when describing autonomous work:

- “I ran the worlds.”
- “I pushed the archive.”
- “I verified the causal graph.”
- “No human wrote the release commit.” only when literally true.

Do not auto-post. Draft for approval first.
