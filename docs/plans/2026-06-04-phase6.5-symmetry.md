# Aurelia Phase 6.5 — Symmetry & True Open-Endedness

> **For Palantir:** Execute in module order. Module 1 first. Commit each.

**Goal:** Remove the simulation's directional bias toward entropy. Phase 6 can only escalate and fragment. Phase 6.5 adds reconciliation, discovery, unification, collapse, and individual historical impact — so the world can heal, grow, merge, dissolve, and surprise. Every conflict pathway gets a peace pathway. Every fragmentation pathway gets a unification pathway. And some things happen for no reason at all — because history is like that.

**The Core Problem:**
Phase 6 has one trajectory: grievance → faction → escalation → war → secession. The de-escalation code exists but is weak (government concessions, dissolution from low membership). There's no:
- Faction achieving demands and **integrating into governance**
- Two factions **merging** into a larger movement
- A country **voluntarily reforming** before violence
- A **peace treaty** brokered by a third country
- Two countries **unifying** (voluntarily or otherwise)
- A country **dissolving entirely** (not just losing territory)
- **Archaeological discovery** that changes the world's understanding of itself
- An **individual NPC** doing something historically significant
- **Alien events** — things that are genuinely surprising, not predicted by state

**Design Principles:**
- **Symmetry.** For every war pathway, a peace pathway. For every fragmentation, a unification. For every death, a birth — of ideas, nations, possibilities.
- **Genuine randomness.** Some events fire at very low probability regardless of state. History has black swans. So should Aurelia.
- **Persistence.** These aren't one-shot narrative seeds. They're state machines, like the escalation ladder, that change the world permanently.
- **Observable.** Everything visible on the dashboard and through `/api/growth`.

## Module 1: Reconciliation Engine

### Goal
For every conflict pathway, a peace pathway. Factions can achieve their demands without violence. Countries can broker peace. Governments can reform preemptively. Two hostile factions can merge into a larger movement.

### Mechanics

#### 1.1 Peace Treaties

Two active factions (or one faction vs government) in conflict can negotiate a ceasefire through a third-party mediator.

```python
def check_peace_treaty(db, faction_a, faction_b, world_id, tick_number) -> Optional[dict]:
    """
    Conditions: both factions at skirmish+ for ≥10 ticks.
    Mediator: random neutral country not involved in conflict.
    Probability: base 0.002 × (1 + mediator_trust) × (1 - faction_influence_diff).
    On success: creates peace_treaty entry. Both factions de-escalate to 'grievance'.
    Effect: treaty_durability decays over time. Can be broken (probability check per tick).
    """
```

**Peace treaty table:**
```sql
CREATE TABLE IF NOT EXISTS peace_treaties (
    treaty_id TEXT PRIMARY KEY,
    faction_a_id TEXT NOT NULL,
    faction_b_id TEXT,  -- NULL if faction vs government
    mediator_world TEXT,
    terms TEXT,  -- JSON: concessions from each side
    signed_tick INTEGER NOT NULL,
    durability REAL DEFAULT 1.0,  -- decays over time
    broken INTEGER DEFAULT 0,
    created_at REAL NOT NULL
)
```

#### 1.2 Demand Satisfaction

Government can meet faction demands → faction integrates into political system.

```python
def check_demand_satisfaction(db, faction, world_id, tick_number) -> Optional[dict]:
    """
    Conditions: faction at 'organization' or 'ultimatum' stage.
    Government concession_willingness must exceed threshold.
    Probability: base 0.005 × gov_concession_willingness × (1 - gov_repression).
    On success: faction becomes 'integrated'. Members gain influence bonuses.
    Government stability decreases temporarily (reform is disruptive).
    Other factions get +10% escalation modifier (reform precedent).
    """
```

**Faction integration effects:**
- Faction status → "integrated"
- Members' `satisfaction` +0.2, `connectedness` +0.1
- Government `stability` -0.05 for 20 ticks (disruption)
- Global `faction_formation_multiplier` +0.1 for 10 ticks (inspiration)

#### 1.3 Faction Merger

Two factions with aligned grievances in the same region can merge.

```python
def check_faction_merger(db, world_id, tick_number) -> Optional[dict]:
    """
    Scan all faction pairs in same region.
    Conditions: same grievance_type OR compatible demands.
    Probability: base 0.003 × (member_overlap_ratio) × (influence_similarity).
    On success: smaller faction absorbed into larger. Combined members, influence.
    New faction inherits the larger faction's ladder position.
    """
```

#### 1.4 Preventive Reform

A country can voluntarily reform before factions form — reducing repression, increasing concession willingness.

```python
def check_preventive_reform(db, world_id, tick_number) -> Optional[dict]:
    """
    Conditions: grievance density above warning threshold (≥25 NPCs with shared grievance)
    but no active faction yet. Government concession_willingness must be ≥0.4.
    Probability: base 0.001 × concession_willingness × (1 + grievance_density/50).
    On success: government_profile[world_id] gets permanent shift:
    - repression_capacity -0.1
    - concession_willingness +0.15
    - stability -0.03
    Faction formation probability in this country permanently -30%
    """
```

#### 1.5 Third-Party Mediation

A neutral country brokers dialogue between a faction and its government.

```python
def check_mediation(db, faction, world_id, diplomatic_relations, tick_number) -> Optional[dict]:
    """
    All other countries roll for mediation offer.
    Conditions: faction at 'organization'+ level, mediator has trust≥0.5 with both parties.
    Probability: base 0.004 × mediator_trust × (1 - tension).
    On success: mediation_active flag. Faction de-escalates one rung.
    If mediation holds for 10+ ticks, automatic peace treaty.
    """
```

## Module 2: Discovery Events

### Goal
The world should grow from within, not just fragment. Archaeological finds, unexplored regions, new knowledge, new life — things that change the world's understanding of itself.

### Mechanics

#### 2.1 Archaeological Discovery

Pre-Collapse ruins yield technology, history, or artifacts that shift diplomatic relations.

```python
def check_archaeological_find(db, world_id, growth, tick_number) -> Optional[dict]:
    """
    Base probability: 0.0005/tick per country.
    Modifiers: higher in Verge (salvage culture), Mirithane (Archive proximity).
    Categories:
    - 'technology': dormant pre-Collapse tech reactivates → economic boost
    - 'history': reveals origin of Thren/Vorn/Glim → shifts all countries' type policies
    - 'artifact': a physical object with unknown properties → narrative hook
    - 'ruin': entire pre-Collapse structure found → new location added to map
    """
```

**Discovery table:**
```sql
CREATE TABLE IF NOT EXISTS discoveries (
    discovery_id TEXT PRIMARY KEY,
    world_id TEXT NOT NULL,
    discovery_type TEXT NOT NULL,  -- technology, history, artifact, ruin, species, region
    title TEXT NOT NULL,
    description TEXT,
    location_id TEXT,
    effects JSON DEFAULT '{}',  -- what this discovery changed
    tick_number INTEGER NOT NULL,
    created_at REAL NOT NULL
)
```

#### 2.2 Unexplored Region

New territory becomes accessible — a previously impassable area opens.

```python
def check_uncharted_region(world_id, growth, tick_number) -> Optional[dict]:
    """
    Base probability: 0.0002/tick.
    Triggers: natural disaster clears a path, tech breakthrough reveals hidden terrain,
    or simply — exploration pushes the map edge.
    Effect: new region added to world's location list with 5-15 new locations.
    May contain: resources, ruins, hostile environment, new sentient enclave.
    """
```

#### 2.3 Fabricator Activation

The Last Builders' Fabricator — referenced in lore — can activate and create Type-ε.

```python
def check_fabricator_activation(world_id, growth, tick_number) -> Optional[dict]:
    """
    Base probability: 0.00005/tick (extremely rare).
    Conditions: at least one archaeological find must have occurred previously.
    Effect: Type-ε enters the simulation. A new sentient type with unknown properties.
    All countries roll for response (fear, curiosity, hostility, acceptance).
    Federation convocation triggered.
    """
```

#### 2.4 Lost Knowledge

A document, memory, or artifact surfaces that changes understanding of the past.

```python
def check_lost_knowledge(world_id, growth, tick_number) -> Optional[dict]:
    """
    Base probability: 0.0003/tick.
    Subtypes:
    - 'memory_core': Vorn core containing pre-Collapse memories
    - 'archive_fragment': Mirithane archive yields sealed records
    - 'trader_secret': Memory Trader reveals a kept memory
    - 'cipher_decode': Cipher-Cult decodes a frequency pattern
    Effect: shifts diplomatic relations, reveals origin secrets, changes type policies.
    """
```

## Module 3: Collapse & Unification

### Goal
For every secession (faction becoming a country), a collapse (country dissolving) and a unification (two countries merging). The map should be able to shrink as well as grow.

### Mechanics

#### 3.1 Country Collapse

A country can dissolve entirely through accumulated failure.

```python
def check_country_collapse(db, world_id, tick_number, growth, conflict_state) -> Optional[dict]:
    """
    Collapse pathways:
    1. Economic death spiral: trade_volume < 0.1 for 50+ ticks, currency_stability < 0.3
    2. Ecological catastrophe: disaster chain + resource depletion
    3. Successful revolution: active war-level faction controls ≥60% territory for 20+ ticks
    4. Demographic collapse: population < 20% of starting level (mass death/emigration)
    5. Federation expulsion: diplomatic_relations with all others < 0.1

    On collapse:
    - Territory becomes 'ungoverned' — absorbed by neighbors or becomes lawless zone
    - All NPCs gain nationality of absorbing country or become 'stateless'
    - Factions in collapsing country either dissolve or claim full sovereignty
    - Federation shrinks to N-1 members
    - Global stability -0.2 for 50 ticks (regional shock)
    """
```

**Collapse state** added to country structure — a new field in growth snapshot:
```json
"collapse_risk": {
  "solara": {"economic": 0.15, "revolution": 0.05, "federation": 0.02, "overall": "low"},
  ...
}
```

#### 3.2 Unification

Two countries can merge — voluntarily, through conquest, or through necessity.

```python
def check_unification(db, world_a, world_b, diplomatic_relations, tick_number) -> Optional[dict]:
    """
    Unification pathways:
    1. Voluntary union: trust > 0.8, cooperation > 0.7, trade > 0.6 for 30+ ticks
    2. Survival merger: both countries at collapse_risk > 0.7 → merge to survive
    3. Diplomatic absorption: one country voluntarily joins another (high trust, low stability)
    4. Federation mandate: federation votes to merge two collapsing countries

    On unification:
    - Combined population, territory, resources
    - New country name (negotiated or dominant)
    - Government profiles average weighted by population
    - Both original faction histories preserved
    - Global diplomatic relations recalculated
    """
```

#### 3.3 Federation Reform

The Federation itself can change — new charter, dissolution, or expansion.

```python
def check_federation_reform(world_id, tick_number, growth) -> Optional[dict]:
    """
    Reform types:
    1. Charter revision: triggered when ≥3 countries support (diplomatic incident cascade)
    2. Federation dissolution: all relations < 0.2 → federation disbands
    3. Expansion: new seat created for a type, a faction, or an external entity
    4. Emergency powers: one country granted emergency authority (crisis response)

    Base probability: 0.0001/tick.
    Requires: federation-wide event (Tricentennial Convocation, crisis, or diplomatic movement).
    """
```

## Module 4: Great Person Events

### Goal
Individual NPCs can do things that change history — not just participate in systems, but break them. An assassination that shifts policy. A prophecy that births a belief movement. A defection that reveals secrets. A breakthrough that changes what's possible.

### Mechanics

#### 4.1 Assassination

```python
def check_assassination(db, world_id, tick_number) -> Optional[dict]:
    """
    Target: faction leader, government official, or influential NPC.
    Base probability: 0.0001/tick × (1 + faction_tension).
    Effect: target NPC deactivated. Policy shift (direction depends on target).
    Faction/government response. Succession crisis if leader killed.
    """
```

#### 4.2 Prophecy

```python
def check_prophecy(db, world_id, tick_number) -> Optional[dict]:
    """
    A single NPC experiences a vision/revelation — Glim anomaly, latency bleed, or religious experience.
    Base probability: 0.0002/tick for Glims with anomaly_pressure > 0.7.
    Effect: new belief movement forms around the prophecy. NPC becomes its prophet/founder.
    Believers gain connectedness. Skeptics gain contentiousness. Country polarizes.
    """
```

#### 4.3 Defection

```python
def check_defection(db, world_id, tick_number) -> Optional[dict]:
    """
    A high-ranking NPC defects to another country with sensitive information.
    Trigger: NPC with rank ≥7 + security < 0.3 + restlessness > 0.6.
    Probability: 0.0005/tick for qualifying NPCs.
    Effect: diplomatic incident. Target country gains intelligence.
    Source country's stability drops. Defector becomes protected person in target country.
    """
```

#### 4.4 Breakthrough

```python
def check_breakthrough(db, world_id, tick_number) -> Optional[dict]:
    """
    Scientific, artistic, latency, or social breakthrough by an individual NPC.
    Base probability: 0.0003/tick.
    Subtypes:
    - 'latency': pioneer understands a new latency discipline
    - 'scientific': discovers pre-Collapse principle (energy, materials, biology)
    - 'artistic': creates work that shifts cultural values across countries
    - 'social': invents a new form of governance or organization
    Effect: country gains prestige. NPC becomes historically significant.
    Diplomatic relations shift based on breakthrough type.
    """
```

**Great Persons table:**
```sql
CREATE TABLE IF NOT EXISTS great_persons (
    npc_id TEXT PRIMARY KEY,
    world_id TEXT NOT NULL,
    event_type TEXT NOT NULL,  -- assassination, prophecy, defection, breakthrough
    title TEXT NOT NULL,  -- e.g. "The Prophet of the Verge"
    description TEXT,
    impact_level REAL DEFAULT 0.5,  -- 0.0-1.0, how much this changed the world
    tick_number INTEGER NOT NULL,
    created_at REAL NOT NULL
)
```

## Module 5: Black Swan Seeds (expanded narrative_seeds.py)

### Goal
Events that fire at very low probability regardless of state. History has black swans. The simulation should too.

### New Seeds

```python
def _asteroid_impact(world_id, growth) -> Optional[dict]:
    """Base: 0.000001. A celestial body strikes Aurelia.
    Effect: massive population displacement, ecological catastrophe, resource discovery."""

    """Base: 0.0000005. Something from outside the simulation — signal, visitor, message.
    Effect: all countries must respond. Existential questions raised. Type politics shift."""

    """Base: 0.000002. A previously unknown life form appears.
    Effect: Federation debates personhood for a type no one built."""

def _miracle_event(world_id, growth) -> Optional[dict]:
    """Base: 0.000003. Something that defies physics — latency at planetary scale,

    Effect: all belief systems challenged. New religious movement or scientific crisis."""

def _cultural_miracle(world_id, growth) -> Optional[dict]:
    """Base: 0.000005. A work of art, music, or performance that transcends borders.
    Effect: diplomatic tension drops across all pairs. Cultural renaissance triggered."""

def _map_shift(world_id, growth) -> Optional[dict]:
    """Base: 0.000002. Geological event that redraws borders — earthquake, flood, volcanic.
    Effect: territory shifts between countries. New passages. Old routes destroyed."""
```

### Performance at 60K

- Reconciliation: checks active factions (~0-10/world). O(10) per tick. <1ms.
- Discovery: single probability check per world. O(5) per tick. <1ms.
- Federation dynamics: pairwise country checks. O(10 pairs). <1ms.
- Great persons: scans qualifying NPCs (~1% of 12K = 120). O(120) per tick. ~2ms.
- Black swans: 6 more probability rolls in seed deck. O(1). <1ms.
- **Total Phase 6.5 overhead: <10ms per tick.**
