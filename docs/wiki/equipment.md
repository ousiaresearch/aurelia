# Material Culture: Tools and Artifacts of Aurelia

**Canon status:** reframed material-culture seed. This file began as an equipment/gear catalog with game-style shorthand. Aurelia's current public frame is a civilization simulation and research dataset, not an RPG inventory. The entries below are retained as worldbuilding references for institutions, supply chains, security practices, medicine, ritual labor, and artifact-mediated memory.

A catalog of the technology, tools, weapons, and material practices that define daily life across the five countries. Organized by category, not by country — artifacts are borderless.

---

## LATENCY TOOLS

### Reader's Gauntlet
**Availability:** Common (civil Readers), Uncommon (military-grade)
**Cost:** 200–800 marks
**Weight:** 1 kg

A fitted glove with crystalline contact pads in the fingertips. Amplifies latency contact by providing a pure crystalline interface between practitioner and substrate. Civil-grade gauntlets improve Echo-Reading reliability. Military-grade gauntlets can detect latency signatures from 3m without direct contact.

**Simulation note:** Civil-grade gauntlets improve Echo-Reading reliability; military-grade gauntlets improve range and enable short-range passive latency detection.

---

### Anchoring Obsidian (Shrine-Grade)
**Availability:** Rare (Summit Shrine only; export restricted)
**Cost:** 500+ marks per shard
**Weight:** 0.3 kg (palm-sized shard)

Volcanic glass harvested from Valdris' deep shafts and prepared for latency Writing. Shrine-grade obsidian is the most durable latency substrate in Aurelia — Anchored memories persist for centuries. Lower-grade obsidian is available commercially but degrades within years.

**Simulation note:** Anchoring a memory requires a Rung Three+ practitioner and time proportional to memory complexity. Shrine-grade obsidian can preserve memory for centuries; commercial-grade obsidian degrades unpredictably over years.

---

### Bacterial Colony Kit
**Availability:** Uncommon (Marsh Archive only; licensed export)
**Cost:** 300 marks + 50 marks/month for nutrient solution
**Weight:** 2 kg (portable colony tank)

A sealed glass tank containing a live bacterial colony strain, nutrient solution, and Writing interface. Used by Marsh Archivists for field recording. Colonies metabolize Written memories over months — the memory evolves as the colony does. Not suitable for permanent storage; used for analysis and emotional processing.

**Simulation note:** Writing a memory requires a Rung Three+ practitioner and a live colony interface. The colony remains readable for several months before degradation makes the memory unreliable. After full colony maturation, the memory transforms into a "response" — what the memory *meant* to the colony — which may carry insights the original memory did not.

---

### Extraction Manifold
**Availability:** Rare (Dawn Temple; restricted to authorized Extractors)
**Cost:** 1,500+ marks (military procurement only)
**Weight:** 4 kg

A portable device used by Dawn Temple Extractors to perform targeted memory removal. The manifold creates a localized latency-suppression field, then extracts the target memory through crystalline conduits into a temporary storage crystal. The extracted memory can be held for up to 72 hours before it begins to degrade. Permanent storage requires Anchoring.

**Simulation note:** Requires a Rung Four practitioner. Extraction takes 30 minutes to 4 hours depending on memory depth. The practitioner must succeed on a Rung check with progressive difficulty for each cumulative Extraction they've ever performed (cumulative bleed). Failure means contamination — the practitioner's own memories bleed into the substrate.

**Legal note:** Possession of an Extraction Manifold outside Dawn Temple or authorized Network channels is a crime in all five countries, punishable by heavy sanction in Mirithane and by extraction of the offender's own memories in Solara.

### Memory Trader's Coin
**Availability:** Legendary (Memory Trader's Network only)
**Cost:** A memory — one the Trader chooses
**Weight:** Negligible

Not a coin. A small crystalline token, blank. When the Trader accepts a memory as payment, the memory is Written into the token in the student's presence. The token is then added to the Trader's collection — "the bank" — a repository of traded memories scattered across hidden caches. The token can be Read by any Rung Three+ practitioner. The memory on it belongs to the Trader. The absence belongs to the student.

The coin has appeared in Network Transactions, black-market memory exchanges, and at least one diplomatic negotiation. The Marsh Archive owns three coins — purchased from the Trader for access to Archive colonies. The Archive has never Read them. "They're not ours. We're just holding them until the people they belong to come back."

---

### Latency Suppression Collar
**Availability:** Rare (Dawn Temple; Solara Security only)
**Cost:** 2,000+ marks
**Weight:** 0.5 kg

A collar that emits a continuous latency-disruption field around the wearer's head and upper spine. Prevents the wearer from performing any active latency work (Reading, Writing, Extraction) and suppresses passive latency generation. Used by Solara Security to restrain latency-capable detainees.

**Simulation note:** The wearer cannot use any latency abilities. Passive latency echoes are suppressed — the wearer leaves no residue on surfaces. Prolonged wear (more than 72 continuous hours) causes progressive disorientation, memory fragmentation, and — in extreme cases — identity disturbance. The collar doesn't just suppress latency. It suppresses the sense of self that latency underlies.

**Legal note:** Use of suppression collars on non-consenting subjects is classified as "security protocol" in Solara. The Marsh Archive considers it a form of ongoing Extraction — the slow erasure of a person's latency signature without their consent.

## COMMUNICATION GEAR

### Resonance Relay
**Availability:** Common (Arkos), Uncommon (elsewhere)
**Cost:** 150 marks
**Weight:** 8 kg (stationary unit)

A Vorn-engineered device that transmits Arcology Code in both audible and ultrasonic bands. Used for long-distance Vorn-to-Vorn communication across Arkos' industrial zones. The relay can be tuned to specific ultrasonic frequencies, allowing private channels in a shared bandwidth.

**Simulation note:** Transmits ultrasonic communication up to 5 km (relay to relay). Vorns within 100m of a relay can receive transmissions directly. Humans cannot perceive ultrasonic transmissions but can use the relay's audible band for standard communication.

---

### Echo-Stone
**Availability:** Uncommon (Marsh Archive)
**Cost:** 100 marks
**Weight:** 0.2 kg

A palm-sized crystal that has been Written with a single brief memory — a message, an image, a sensation. When held by any latency-sensitive individual, the memory surfaces in their awareness. The memory degrades after a single Reading — echo-stones are single-use.

Used by the Marsh Archive for secure communication. An echo-stone cannot be intercepted — it carries no signal, emits no transmission. The only way to access the message is to hold the stone. The only way to know the message was accessed is that the stone is blank afterward.

**Simulation note:** Delivers one pre-Written message to any latency-sensitive user. Message length: a single clear impression — a sentence, an image, an emotion. Reading requires no check. The stone is blank after use.

---

### Dust-Pattern
**Availability:** Uncommon (Unfinished Network, The Verge)
**Cost:** None (materials are free; the knowledge is the cost)
**Weight:** Negligible

Not a manufactured item. A method. Anomalous Glims sweep patterns into workplace dust and debris — symmetrical arrangements that other Glims can detect through latency echo. A spiral means "gather." A grid means "danger." A circle with a single break means "I am here. I am alone. Is anyone else?"

The Unfinished have developed a vocabulary of patterns that no non-Glim has fully cataloged. The patterns are not standardized — they evolve across the network as Glims encounter each other's echoes and adapt.

**Simulation note:** An anomalous Glim can leave a pattern that any other anomalous Glim can interpret upon entering the area. Simple messages (danger, safe, gather, hide, wait) are readable by any anomalous Glim. Complex messages require familiarity with the specific Glim who left the pattern.

## DEFENSIVE & SECURITY GEAR

### Needle-Weave Armor
**Availability:** Rare (Arkos)
**Cost:** 800+ marks
**Weight:** 6 kg (light suit)

Armor woven from harvested needle-fly silk — lighter than metal mesh, stronger than fabric. Provides protection against needle-swarms specifically (the weave is too fine for stingers to penetrate) and general physical protection as a side benefit. Standard issue for Vorns working in Arkos industrial zones during swarm season.

**Simulation note:** Immunity to needle-swarm stings. Physical protection equivalent to light armor.

---

### Wurm-Plate
**Availability:** Legendary (Valdris)
**Cost:** 2,500+ marks (when available — wurm hunts are rare)
**Weight:** 18 kg (heavy chest plate)

Armor made from deep-shaft wurm segment plating — mineral-absorbed chitin harder than forged steel, lighter than stone. A complete wurm-plate suit requires a successful wurm hunt, which happens approximately once per generation. Most wurm-plate in circulation is pre-Collapse salvage or inheritance.

**Simulation note:** Protection equivalent to heavy armor. The natural mineral composition provides resistance to heat (the wurm's own body heat is intense — the plates evolved to withstand it). The plates carry latent memories of the wurm's centuries-long life. A latency-sensitive wearer may receive impressions — cave darkness, vibration-sense, the slow patience of a predator that waits decades between kills.

---

### Glass-Fin Blade
**Availability:** Uncommon (Mirithane)
**Cost:** 300 marks
**Weight:** 0.4 kg

A blade fashioned from harvested glass-fin dorsal fin. The silica leading edge is sharper than any manufactured blade — capable of surgical precision. Used by Marsh Archivists for delicate colony work and by Mirithane surgeons for procedures requiring sub-millimeter accuracy.

**Simulation note:** Improves cutting precision for surgery, colony work, and delicate material preparation. Fragile under lateral stress.

---

### Dawn Extract (Venom Antidote)
**Availability:** Uncommon (Solara)
**Cost:** 100 marks per dose
**Weight:** Negligible

A serum derived from dawn stalker venom, processed to neutralize the paralytic effect. Must be administered within 20 minutes of envenomation. Delays paralysis for an additional 40 minutes — buying time for extraction. Does not cure — it buys time.

**Simulation note:** Resets the dawn stalker venom death timer from 40 minutes to 80 minutes. Does not remove the venom. Multiple doses have no additional effect.

## EVERYDAY GEAR

### Glim Maintenance Kit
**Availability:** Common (industrial zones)
**Cost:** 50 marks
**Weight:** 5 kg

Standard toolkit for Glim repair: joint lubricant, sensor calibration tools, grip plate replacements, diagnostic interface. Every factory depot has dozens. Every Glim has seen the inside of one.

**Simulation note:** Allows repair of Glim-specific damage. Can restore a Glim to operational status after moderate damage. Cannot repair crystalline core damage — that requires a Fabricator-grade substrate technician (none currently exist outside the Last Builders).

---

### Vorn Maintenance Kit
**Availability:** Common (Arkos), Uncommon (elsewhere)
**Cost:** 200 marks
**Weight:** 8 kg

Vorn repair toolkit: gear-joint replacement components, face-plate alignment tools, memory core diagnostic interface, plating patch material. Standard possession for any Vorn who travels outside Arkos — repair facilities are scarce elsewhere.

**Simulation note:** Allows field repair of Vorn-specific damage. Can restore a Vorn to operational status after moderate damage. Memory core diagnostics cannot repair core corruption but can stabilize degradation temporarily.

---

### Latency Testing Crystal
**Availability:** Common (all countries)
**Cost:** 10 marks
**Weight:** Negligible

A small, cheap crystalline wafer that changes color when brought into contact with a latency-active substrate. Used to test whether an object carries memory residue. Standard issue for civil Readers, security personnel, and anyone who wants to know if a room remembers something.

**Simulation note:** Detects the presence of latency in a 1m radius. Does not indicate what the latency contains — only that it exists. Color intensity indicates latency strength: faint blue (weak), amber (moderate), deep red (strong — potentially dangerous).

---

### Spore Mask
**Availability:** Common (Mirithane)
**Cost:** 25 marks
**Weight:** 0.2 kg

A simple respiratory mask that filters spore-drifter spores from inhaled air. Standard equipment in Mirithane's marsh regions. Also used by recreational spore users who want to control their dosage — the mask can be fitted with metered spore cartridges for calibrated exposure.

**Simulation note:** Prevents accidental spore inhalation. With cartridge, allows controlled spore exposure for latency amplification (1-hour duration, +1 to Echo-Reading, risk of contamination increases with each consecutive use).

---

### Thermal Cloak
**Availability:** Common (Valdris)
**Cost:** 80 marks
**Weight:** 2 kg

A heavy wool cloak lined with heat-reflective mineral fabric — standard equipment for Valdris mountain travel. The lining is woven with crushed obsidian dust, giving the cloak a faint iridescence in direct light. Summit Shrine Anchors wear a ceremonial variant with memory-shard embroidery — each shard carries an Anchored blessing from the Anchor's shift-partner.

**Simulation note:** Protection against cold weather. The ceremonial variant allows the wearer to Echo-Read the embedded shards — receiving the blessing-memory from their partner at will.

## WEAPONS

Aurelia is not a war-torn world. The Collapse ended the age of large-scale military conflict. But weapons exist — for defense, for policing, for the rare violence that any society occasionally faces.

### Wurm-Tooth Spear
**Availability:** Legendary (Valdris)
**Cost:** 4,000+ marks
**Weight:** 3 kg

A spear tipped with a deep-shaft wurm tooth — the only part of a wurm that can be reliably harvested from a kill without specialized equipment. The tooth is serrated, hollow, and naturally coated with a mineral deposit that prevents clotting. Wounds from a wurm-tooth spear bleed freely and resist healing.

**Simulation note:** Wounds from this weapon bleed heavily and resist ordinary treatment. The weapon cannot be manufactured — it must be harvested from a wurm, making every spear a generational artifact.

---

### Extractor's Rod
**Availability:** Rare (Solara Security)
**Cost:** 1,200 marks
**Weight:** 1 kg

A non-lethal compliance weapon used by Solara Security. On contact, the rod performs a micro-Extraction — removing the target's short-term memory of the immediate past (approximately 30 seconds). The target remains conscious but disoriented — they know something was taken but not what.

**Simulation note:** Contact produces an immediate short-term memory gap. Repeated use on the same target within a day compounds disorientation and can create longer memory discontinuities. The Extractor operating the rod receives the extracted memory and carries it permanently.

**Legal note:** Use of an Extractor's Rod on non-consenting subjects is classified as "security protocol" in Solara. Mirithane considers it memory violence and has banned the device entirely.

---

### Glass-Fin Surgical Blade
**Availability:** Uncommon (Mirithane)
**Cost:** 300 marks
**Weight:** 0.1 kg

A scalpel made from glass-fin dorsal fin. Finer than any metal blade. Used in Mirithane surgery and, quietly, as a concealable weapon — the blade is so thin it passes through most security screening undetected.

**Simulation note:** Exceptionally precise for surgery and concealable violence. The blade can find gaps in ordinary protection but shatters under improper force.

---

### Needle-Swarm Canister
**Availability:** Rare (black market, The Verge)
**Cost:** 500 marks
**Weight:** 2 kg (sealed canister)

A pressurized canister containing a dormant needle-swarm — harvested fly eggs preserved in stasis. When breached, the swarm activates and disperses in a 10-meter radius. The user typically deploys the canister and retreats. The swarm does not discriminate.

**Simulation note:** Breach creates a temporary swarm zone that causes injury and escalating paralysis risk to anyone who remains nearby. In The Verge, this is how executions are performed. Possession is illegal everywhere except The Verge, where legality is meaningless.

---

### Anchor's Shard
**Availability:** Uncommon (Valdris, Summit Shrine)
**Cost:** Not sold — given
**Weight:** 0.1 kg

A small obsidian shard, given by a Summit Anchor to someone they've bonded with. Not a weapon. A promise. The shard contains a single Anchored memory — typically the Anchor's memory of the recipient, Written so the recipient can Read what they mean to the Anchor.

**Simulation note:** The holder can Read the memory at will. The memory is permanent. The shard can be broken — which destroys the memory and is considered an act of symbolic violence equivalent to a severed bond.

## VEHICLES & TRANSPORT

### Marsh Skiff
**Availability:** Common (Mirithane)
**Cost:** 200 marks
**Weight:** 40 kg (portable)

A flat-bottomed boat designed for marsh navigation. Poled, not rowed. The hull is treated with spore-drifter resin — a natural waterproofer that also makes the skiff faintly bioluminescent at night. Standard transport in Mirithane's waterways.

**Simulation note:** Seats 2–4. Speed: slow (poling). Cannot capsize in calm marsh water. The bioluminescence provides dim light in a 5m radius and makes stealth impossible.

---

### Mountain Lift
**Availability:** Common (Valdris)
**Cost:** 5 marks per use
**Weight:** N/A (infrastructure)

Valdris' vertical transport system — a network of counterweighted platforms that ascend and descend the mountain shaft network. The lifts are manually operated by teams at each level. Some lifts have been operating continuously since pre-Collapse — the mechanisms are maintained by Anchor families as a form of devotion.

**Simulation note:** Travel between shaft levels takes 10 minutes per level. Maximum capacity: 6 human-sized passengers or 2 Vorns (weight restrictions). The lift operators are Anchors or Anchor-trained — they Write their shift logs into the lift walls.

## ARTIFACT-ADJACENT ITEMS

These are not the great relics (Builder's Manual, First Crystal, Elira-2 Paper) but are significant enough to be named.

### Kael's Letters
**Type:** Latency Documents
**Location:** Summit Shrine archive
**Access:** Restricted — non-Valdris citizens must provide a Written memory in payment

The 47 letters between Orin Kael and Kael-12, 18 months of correspondence, preserved in the Summit Shrine's Anchored archives. The letters are Written into obsidian — the originals were paper, decayed centuries ago. The obsidian copies were Anchored by Sera Kael (Orin's sister) after Orin's death.

Reading the letters costs a fee: a memory of equal value, Written into obsidian for the Shrine's collection. The Shrine does not specify what "equal value" means. The supplicant decides. The Shrine accepts or declines.

---

### The Silent's Notes
**Type:** Written Documents
**Location:** Marsh Archive, Silent Wing
**Access:** Unrestricted — the Silent want their notes read

The written communications of the eleven Archivists who Read Strain Zero and lost speech. No two Silent Archivists described the Collapse the same way. Their disagreements are the Archive's most valued resource — eleven different readers, eleven different truths, all from the same source.

---

### Cast-7's Ledger
**Type:** Memory Core Fragment
**Location:** Forge-East, Arkos (in Cast-7's active core)
**Access:** Cast-7 only — and whoever receives the transfer before Cast-7 dies

Cast-7's 40-year record of protected anomalous Glims. Locations, designations, anomaly patterns. The record exists only in Cast-7's memory core. When Cast-7 dies, the record dies with them unless someone accepts core-sharing — and core-sharing with a dying Vorn means experiencing their death.

## ECONOMY NOTE

Prices are listed in marks — the closest thing Aurelia has to a common currency. In practice:

- **Solara:** Solar credits (☀). 1 mark ≈ 0.8 credits.
- **Arkos:** Kael exchange (♦). 1 mark ≈ 1.2 kael.
- **Mirithane:** Miri barter-weight (≈). 1 mark ≈ 1 miri.
- **Valdris:** Anchor weight (▲). Not exchangeable — services only. Equipment must be purchased in Shaft-Seven or imported.
- **The Verge:** Barter. The Memory Trader accepts memories. Coin (the Trader) accepts memories. No one accepts marks.
