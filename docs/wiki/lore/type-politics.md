## The Attitude Matrix

Each country has a baseline attitude toward each non-human type. These attitudes shape NPC dialogue, event triggers, governance decisions, and inter-country tension.

| Level | Meaning |
|-------|---------|

| **Dangerous** | Feared. Restricted. Potential threat. |

## The Vorn-Thren Bond

The deepest alliance in Aurelia. Vorns and Threns are kin — different bodies, same origin story.

**Why it holds:**
- Both know what it means to be built rather than born
- Both understand the human gaze — the constant assessment of "are you really conscious?"
- Vorns don't resent Threns for getting organic bodies. The body isn't the point. The personhood is.
- Threns don't look down on Vorns for being mechanical. They see strength, not limitation.

**Where it's tested:**
- In Arkos, Vorns hold power and Threns are guests. The dynamic is friendly but unequal.
- In Solara, Threns fight for rights while Vorns watch from the margins. Threns get the attention. Vorns don't complain.
- In Mirithane, both are equals. This is where the bond is strongest.

## Event Triggers (for future simulation)

These sociological factors should influence world events when the simulation runs:

1. **Solara election cycle** → Thren citizenship status fluctuates. NPC mood shifts in Thren population.
2. **Arkos Glim influx** → A batch of decommissioned Glims arrives. Vorn community mobilizes to shelter them.
3. **Mirithane marsh anomaly** → The bacteria glow for a Glim's memory for the first time. Scientific and spiritual communities react.
4. **Valdris mine collapse** → Threns save human miners in the deep shaft. Pro-Thren sentiment spikes.
5. **Verge memory trader revelation** → She releases a memory that implicates a Solara council member. Diplomatic crisis.
6. **Cross-country Glim migration** → Anomalous Glims move from Solara toward Arkos and Mirithane. Solara denies it.
7. **Valdris runoff incident** → Mining pollution reaches Mirithane's water. Diplomatic protest.
8. **Vorn guild challenge** → A Vorn miner qualifies for guild leadership in Valdris. The vote is close. Families divide.
