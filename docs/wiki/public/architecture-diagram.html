Aurelia Federation — System Architecture

Federated world simulation · 5 country-states · 11 autonomous daemons · 30-min tick cycle

Federation Layer

  • • 1 Coordinator on port 9001
  • • HTTP server with 7 API endpoints
  • • Dark HTML dashboard (auto-refresh 30s)
  • • World registry + health tracking
  • • Cross-world currency exchange rates

Simulation Layer

  • • 5 country daemons, 6 agent daemons
  • • 30-minute tick interval (2:1 time ratio)
  • • Per-tick: time, NPC AI, currency, ecology
  • • SQLite WAL mode, one DB per world
  • • PID-based crash recovery, SIGTERM clean

Data Layer

  • • ~20,000 lines of shared engine code
  • • 600 NPCs with schedules & social graphs
  • • 4 currencies with live exchange rates
  • • Zero external dependencies (stdlib only)
  • • Python 3.13 + YAML configs
